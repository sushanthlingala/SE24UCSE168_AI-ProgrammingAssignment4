# AI Programming Assignment 4 – CS2201

This repository currently contains the codes and documentations for the following questions detailed in our Programming Assignment 4 for the CS-2201 course:

    1. Implement the Map Coloring problem using CSP for the seven principal states and territories of Australia.
    2. Apply the Map Coloring problem to the 33 districts of Telangana.
    3. Implement the Sudoku puzzle using Constraint Satisfaction Problem (CSP) techniques.
    4. Implement the Crypt-arithmetic puzzle (TWO + TWO = FOUR) using CSP.

The codes and related documentation (if required) have been posted within the respective folders (named as per the question).

To clone this entire repository, please use:
git clone https://github.com/sushanthlingala/SE24UCSE168_AI-ProgrammingAssignment4.git

---

## Repository Structure

The repository is organized into the following folders:

- **Q1-MapColouring_Australia**  
  Contains the implementation of the Map Coloring problem for the seven principal states and territories of Australia using CSP techniques such as backtracking, forward checking, and constraint propagation.

- **Q2-MapColouring_Telangana**  
  Contains the implementation of the Map Coloring problem for the 33 districts of Telangana.  
  The adjacency relationships are stored externally (JSON) to separate data from logic and improve scalability.

- **Q3-Sudoku**  
  Contains the implementation of a Sudoku solver using CSP techniques including AC-3 (arc consistency) and backtracking search.

- **Q4-Crypt_Analysis**  
  Contains the implementation of the crypt-arithmetic puzzle solver (TWO + TWO = FOUR) using CSP formulation with constraints such as Alldiff and column-wise arithmetic consistency.
